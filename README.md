# সালামি স্টেশন 🌙 — Salami Station

![সালামি স্টেশন Preview](https://s6.imgcdn.dev/YvAMuL.png)

ঈদের সময় সালামি চাওয়াটা একটু awkward লাগে, তাই না? সরাসরি বলতে গেলে লজ্জা লাগে, না বললে পকেট খালি থাকে। এই সমস্যা সমাধানের জন্যই বানিয়েছি **সালামি স্টেশন**।

লিংক বানাও, পাঠাও, সালামি পাও। ব্যস।

**👉 [salami-station.vercel.app](https://salami-station.vercel.app/)**

| :--- | :--- |

### 🛠 কীভাবে কাজ করে

তোমার নাম, বিকাশ নম্বর আর একটা মজার কারণ দাও — অ্যাপ একটা কার্ড বানিয়ে দেবে। সেই কার্ডের লিংক দুলাভাই বা বড় ভাই-বোনদের পাঠিয়ে দাও WhatsApp-এ। বাকিটা তাদের বিবেকের উপর।

---

### 🗺️ রুটগুলো (Routes)

| রুট | কাজ |
|-----|-----|
| `/` | হোমপেজ — প্রজেক্টের পরিচয়, ডেমো কার্ড দেখায় |
| `/getsalami` | কাস্টম সালামি কার্ড তৈরি এবং শেয়ারযোগ্য লিংক জেনারেশন |
| `/gift-founder` | প্রজেক্টের ফাউন্ডারকে (আমাকে) সালামি দেওয়ার বিশেষ পেজ |
| `/request/[id]` | ইউনিক সালামি রিকোয়েস্ট পেজ — যা শেয়ার করা হয় |

### 💻 টেক স্ট্যাক (Tech Stack)

- **Next.js** — App Router এবং Server Actions ব্যবহার করা হয়েছে
- **MongoDB** — রিকোয়েস্টগুলো ডাটাবেসে সেভ করার জন্য
- **Tailwind CSS** — আধুনিক ও রেসপনসিভ ইউআই ডিজাইনের জন্য
- **Vercel** — সুপার-ফাস্ট ডেপ্লয়মেন্টের জন্য
