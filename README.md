# সালামি স্টেশন 🌙

![সালামি স্টেশন Preview](https://s6.imgcdn.dev/YvAMuL.png)

ঈদের সময় সালামি চাওয়াটা একটু awkward লাগে, তাই না? সরাসরি বলতে গেলে লজ্জা লাগে, না বললে পকেট খালি থাকে। এই সমস্যা সমাধানের জন্যই বানিয়েছি **সালামি স্টেশন**।

লিংক বানাও, পাঠাও, সালামি পাও। ব্যস।

**👉 [salami-station.vercel.app](https://salami-station.vercel.app/)**

---

### কীভাবে কাজ করে

তোমার নাম, বিকাশ নম্বর আর একটা মজার কারণ দাও — অ্যাপ একটা কার্ড বানিয়ে দেবে। সেই কার্ডের লিংক দুলাভাইকে পাঠাও WhatsApp-এ। বাকিটা তার বিবেকের উপর।

---

### রুটগুলো

| রুট | কাজ |
|-----|-----|
| `/` | হোমপেজ — প্রজেক্টের পরিচয়, ডেমো কার্ড দেখায় |
| `/getsalami` | এখানে নাম, বিকাশ নম্বর আর কারণ দিলে কাস্টম সালামি কার্ড তৈরি হয় এবং শেয়ারযোগ্য লিংক পাওয়া যায় |
| `/gift-founder` | প্রজেক্টের founder-কে সালামি দেওয়ার পেজ |
| `/request/[id]` | প্রতিটা সালামি রিকোয়েস্টের ইউনিক পেজ — এই লিংকটাই আত্মীয়দের পাঠানো হয় |

---

### টেক স্ট্যাক

- **Next.js** — App Router আর Server Actions দিয়ে পুরো অ্যাপ বানানো
- **MongoDB** — সালামি রিকোয়েস্টগুলো এখানে সেভ হয়
- **Tailwind CSS** — স্টাইলিং
- **Vercel** — হোস্টিং ও ডেপ্লয়মেন্ট
